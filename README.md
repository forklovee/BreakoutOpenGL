# Breakout Clone in OpenGL
Breakout clone made by following [learnopengl.com tutorial](https://learnopengl.com/In-Practice/2D-Game/Breakout).

I've modified it to fit modern c++ style.

# Building & Running
Build with cmake in project directory.
```sh
cmake -S . -B build
```
Run executable "Breakout" from the "build" directory.

# Controls
A - Move left

D - Move right

Space - Launch the ball
